🏠 AI Interior Design System (Figma UI/UX Prototype)

This project is a Software Engineering (SE) based UI/UX prototype designed in Figma, focused on building an AI-powered interior design platform. The system allows users to transform their room designs intelligently using AI technology.

🚀 Project Overview:

The platform enables users to:

Upload images of their rooms
Apply AI-based design transformations
Instantly visualize modern interior layouts

This project demonstrates the concept of integrating AI with user-friendly design interfaces to simplify home interior planning.

🎨 Key Features:
🔐 User Authentication System
→ Secure login/signup functionality
🖼️ Room Image Upload
→ Users can upload their room photos
🤖 AI-Based Interior Design
→ System generates redesigned room concepts using AI models
❤️ Wishlist Feature
→ Users can save their favorite designs
💾 Image Saving & Management
→ Designs can be stored for later use
⬇️ High-Quality Downloads
→ Users can download final designs in HD quality
💳 Subscription Model
→ Premium features like HD downloads and unlimited saves
🧠 Purpose of the Project:
To design a real-world SaaS-based AI application interface
To understand UI/UX principles and user flow design
To simulate how AI can enhance user experience in interior design
🛠️ Tools Used:
Figma (UI/UX Design & Prototyping)
